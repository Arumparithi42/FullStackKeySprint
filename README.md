# KeySprint

Small React + Express typing-test project (still in development process).
[Live Demo](https://full-stack-key-sprint.vercel.app) 

## About

KeySprint is an interactive typing test web application that helps users evaluate and improve their typing speed, accuracy, and efficiency. It provides real-time performance metrics and a responsive platform for continuous practice.

## Features

1. Register and Login
2. Different time test (30s, 60s, 120s)
3. Tracks Accuracy, Words Per Minute(WPM), Weakest letter
4. Profile Editing

## Tech Stacks
1. HTML
2. CSS
3. javaScript
4. React.js (Vite)
5. Express.js
6. Node.js 
7. MongoDB (Mongoose)

## Prerequisites

- Node.js and npm (v16+ recommended)
- MongoDB installed locally (or use a hosted MongoDB URI)

## Frontend (run locally)

1. Install dependencies:

```powershell
npm install
```

2. Start dev server (opens your system browser automatically):

```powershell
npm run dev
```

## Backend (run locally)

1. Install backend dependencies:

```powershell
npm install
```

2. Start the backend server:

```powershell
node server.js
```
